### 대운 (역행, 대운수 6) — **해결 완료(2026-06-11): 시작 나이 일치**
- 간지 순서 3소스 동일: 己酉 戊申 丁未 丙午 乙巳 甲辰 癸卯 壬寅 辛丑 庚子.
- **원인 규명**: lunar-python `getStartYear()=6`(대운수=만나이, 레퍼런스와 일치)은 맞았으나, 엔진이 각
  대운 `start_age`에 lunar-python `getStartAge()`를 그대로 썼는데 이 값은 起運 캘린더연도(2004)의
  **중국식 세는나이(虚岁)=8**이라 +2 어긋남. 虚岁 오프셋은 케이스마다 +1~2로 달라짐(골든 2000-01-01은 +1).
- **수정**: `start_age = 대운수(getStartYear) + 10*순번` 으로 도출(calc/myeongni.py). 이제 6/16/26/36 +
  daewoon_count=6 내부 정합. 근거: 한국 만세력 관행 = 대운수=만나이=시작나이(brunch saju6969·KNS 31강·역학동).
  회귀: test_shinsal.test_case1_daewoon_start_age, test_p2.test_daewoon_start_age_uses_daewoon_count.
- 세운: 2026년 = 丙午年(세운). 엔진 대운 丁未 구간(2024~2034)에 정합 — 모순 없음(丙午는 세운, 대운 아님).
