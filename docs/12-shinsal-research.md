# 12. 신살(神煞) 커버리지 검증·조사 + 채택 결정

> 최초 작성: 2026-06-11. 계기: docs/11 케이스#1 대조에서 신살 커버리지 간극 확인
> (엔진 7종 vs 포스텔러 13종+12신살+공망). 운영자 지시: "어떤 만세력이 가장 정확한지
> 불명확 — 제대로 조사해서 더 좋은 방향으로." 본 문서가 신살 보강의 SSOT.
> 표·기준 출처는 §4 신뢰도 등급으로 분류. 채택 결정은 docs/03 §1 결정표에 행 추가.

## 0. 결론 요약 (3줄)

1. **간극의 원인은 "정확도 차이"가 아니라 선택 차이다.** 유명 만세력이 "더 많은" 이유는
   (a) 채택 범위(앱은 길신·살 수십 종 노출, 엔진은 핵심 7종), (b) 표시 구조(앱=기둥별 per-pillar,
   엔진=사주 전체 평탄 목록), (c) 기준 축(앱=년지+일지 양축 병기, 엔진=일지 단축).
2. **엔진의 7종 표는 대부분 검증상 정확(A등급)하다.** 천을귀인·도화·역마·화개·백호는 고전·현대
   출처 일치. 단 **괴강 구성만 고전 4주설과 불일치**(엔진은 戊戌 누락·壬戌 포함) → 결정 필요(§3).
3. **신살은 풀이에 "보조 색채"를 주지만 길흉 판정의 본체가 아니다.** 명리 본류(자평)는 격국·억부가
   본체이고 신살은 부차(절대규칙 9·11·12). 따라서 보강 방향 = 풀이 섹션에 매핑 가능하고 표가
   검증되는 길신 위주 선별 채택 + §12 비단정 문안. "많이 넣기"가 목적이 아니다.

## 1. 간극이 왜 있는가 (케이스#1 역산 근거)

케이스#1(1997-10-27 09:46 서울 남, 乙巳/壬寅/庚戌/丁丑) 포스텔러 vs 엔진 차이를 4분류:

| 분류 | 내용 | 사례 |
|---|---|---|
| (a) 채택 범위 | 앱은 길신·살을 풍부하게 노출(마케팅·정보량). 엔진은 핵심 7종만 | 금여성·문창·학당·암록·천주귀인·태극귀인·천문성·고신살을 앱은 표시, 엔진 미산출 |
| (b) 표시 구조 | 앱=기둥별(시/일/월/년 각 칸에 해당 신살). 엔진=사주 전체 평탄 list[str] | 앱: "시주 천을귀인·역마 / 일주 문창·학당·암록…" / 엔진: ["천을귀인","화개살","백호"] |
| (c) 기준 축 | 앱=년지+일지 양축 병기. 엔진=일지 단축(삼합계만) | 화개 — 앱은 丑(년지)·戌(일지) 양쪽, 엔진은 일지 寅午戌국 戌만 |
| (d) 적용 범위 | 괴강 — 엔진 일주 한정 4주설, 앱 월주 庚戌에도 표시(전 기둥) | |

핵심: 위 차이는 전부 **학파·표시 정책 선택**이며, 어느 쪽이 "틀린" 게 아니다. 신살 종수가 많다고
풀이가 더 정확한 것이 아니다 — 오히려 검증 안 된 표·과잉 나열은 노이즈·과대해석 위험(§12 충돌).

## 3. 채택 결정 (Phase B 구현 대상)

### 3-1. 채택 (A·B등급 + 풀이 섹션 매핑 가능)

| 신살 | 풀이 섹션 매핑 | 비고 |
|---|---|---|
| 천을귀인(유지) | character·관계 | 기존 |
| 도화·역마·화개(유지) | 관계·이동·적성 | 기존, **기준 축 정책화**(아래) |
| 양인·괴강·백호(유지) | health(에너지 그룹) | 기존, 괴강 구성 결정 후 |
| 문창귀인 | 학업·직업·적성 | 신규, A등급 |
| 금여 | 재물·배우자 | 신규, A등급 |
| 암록 | 재물(숨은 조력) | 신규, A등급 |
| 학당귀인 | 학업·적성 | 신규, B(장생지설 채택·note) |
| 태극귀인 | 총평(복록 색채) | 신규, B |
| 천문성 | 적성(종교·철학·역학·의료·법조) | 신규, B |
| 고신살·과숙살 | 관계(고독·만혼 색채) | 신규, B, §12 비단정 강조 |
| 12신살 12종 | 기둥별 영역 색채 | 신규, 축 정책 확정 후 |
| 공망 | 해당 십성·육친 비움(비단정) | 신규, 자체 산술(A) |

### 3-2. 보류 (C등급 — 미채택, 향후 재평가)

- **천주귀인**: 산출표 미확정 → 1차 출처 확보 전 미채택.
- **음간 양인(乙辰 등)**: 출처 분기(乙→辰 vs 乙→寅), 본령(양간)만 유지.
- **역마 사생지 느슨표기**: 권위 출처 미확인. 엔진은 삼합 충지 정설 유지(포스텔러식 사생지 표기와
  다를 수 있음 → known-diff 후보).

### 3-3. 결정 필요 항목 (docs/03 §1 행 추가 — 운영자 확정)

1. **괴강 구성**: 엔진 현행 {庚辰,庚戌,壬辰,壬戌}(현대 만세력 통용) vs 고전 4주설 {戊戌,庚辰,庚戌,壬辰}.
   → 현대 통용형 유지 + note 권장(壬戌은 백호와 중복, 학파별 분류차).
2. **삼합계 기준 축**(도화·역마·화개·12신살): 일지 단축(현행) / 년지+일지 병기 / 년지.
   → 포스텔러 등 다수 앱 = 양축 병기. 권장: **년지+일지 병기**(앱 정합 + basis로 어느 축인지 표기).
3. **공망 표기**: 일주 기준 본령 + 년주 병기 여부. → 권장: 일주·년주 둘 다 산출, 표기는 일주 우선.

### 3-4. known-diff (포스텔러와 의도적 차이 — 문서화)

- 역마 사생지 느슨표기 미채택(엔진=삼합 충지 정설).
- 음간 양인 미산출.
- 천주귀인 미산출(표 미검증).
- 신살 종수 자체(엔진=선별 채택, 앱=다수 노출) — 정확도가 아닌 편집 정책 차이.

## 4. 검증 신뢰도 등급

- **A (고·현대 출처 일치, 엔진값 검증됨)**: 천을귀인, 문창귀인, 12신살 배치표, 역마(삼합 충지),
  공망(순중공망), 백호 7종, 금여, 암록, 양인(양간).
- **B (본령 명확, 변형·기준축 분기 존재)**: 학당귀인(庚→亥 등 장생 불일치), 12신살 기준 축
  (년지/일지 분기가 정설), 도화·화개 양축 병기, 고신·과숙(1차 권위 약함), 태극귀인(戊己 세부 변형).
- **C (분기 크거나 미확정)**: 괴강(4/5/6주설), 천주귀인(표 미확보), 천문성(강·약 구분 상이),
  역마 사생지 느슨표기(권위 미확인), 포스텔러 12신살 축(캡처 재검토 필요).

## 5. 풀이 디테일 기여도 평가 (운영자 질문: "확실하게 디테일을 주는가")

- **준다 — 단, 보조 색채로서.** 길신(문창·학당→학업, 금여·암록→재물, 천문→적성, 고신·과숙→관계)은
  해당 섹션에 구체 결을 더해 풀이를 풍부하게 한다. 12신살·공망은 기둥별 영역 색채를 준다.
- **그러나 본체는 아니다.** 자평 명리에서 격국·억부·십성·오행이 풀이의 뼈대이고, 신살은 살을 붙이는
  보조다(절대규칙 9). 신살만으로 길흉·시기를 단정하면 §11·§12 위반.
- **과잉의 함정.** 20~30종을 전부 나열하면 정보량은 늘지만 가독성·신뢰도는 떨어지고 과대해석을
  부른다. → 보강 설계는 **섹션별 상한 + 그룹 요약 + 비단정 문안**(Phase C). 검증 안 된 표(C등급)는
  넣지 않는 것이 "더 정확한" 방향.
