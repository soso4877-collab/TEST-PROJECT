# 01. Product Context

> 최초 작성: 2026-06-10

## 상품 정의
- 명리학(메인) + 자미두수(보완) 종합형 PDF 리포트. 1건당 고객 맞춤 생성.
- 입력: 이름(닉네임 가능), 생년월일, 양/음력(윤달 포함), 생시(미상 허용), 출생지, 성별, 고민/질문 텍스트.
- 출력: 검수 완료된 tagged PDF(텍스트레이어·아웃라인), 운영자가 채널로 수동 전달.

## 운영 조건 (확정)
- 관리자(운영자 1인) 검수 후 발송. 검수 없는 자동 발송 금지(상태 머신으로 물리 차단).
- 발송 채널: 당근/카카오 채팅 수동 전달(MVP). 자동 발송은 보류.
- 계산은 deterministic 엔진(LLM 금지). 본문 대부분 규칙 기반 NLG. LLM은 4구간만(질문 분류/통합 해석/질문 답변/마지막 조언) 공식 API로.

## 운영자 결정 사항 (2026-06-10 확정)
1. 자미두수 윤달 산입 기준: **15일 분할법** (윤달 1~15일생=본월, 16일 이후=익월). 리포트에 기준 고지 자동 삽입.
2. 고지 문구 수위: **감수 명시형** — "자동 분석 도구로 산출하고 운영자가 직접 검수·감수한 자료" 수준으로 사실 그대로 고지.
