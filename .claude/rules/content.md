# 콘텐츠 레이어 규칙 (content/)

- 모든 본문(룰 생성·LLM 생성·관리자 수정)은 3단 가드 통과 전 사용 금지: safe_lint(§12 안전표현) → factcheck(간지·별 토큰) → trace(그라운딩). 실패 시 룰 원문 폴백.
- 사실 슬롯 외 새 사실(간지·별·수치·사건) 생성 금지. factcheck allowed_tokens 확장은 계산 데이터 추가와 동시에만.
- 명리x자미 상충 문구: "모순/틀림" 금지, 명리 우선 + 층위 차이 재서술 패턴 사용 (docs/03 §5).
- 윤달 출생 고지("산입 기준: 15일 분할법")·시진 불명 고지(시주 추정) 자동 삽입 누락 금지.
- health 섹션: 의료 단정 절대 금지, "의료 전문가 상의" 문구 고정. 투자·법률 단정 금지.
- 문형 다양화는 _pick(md5 결정론) 유지 — 같은 입력은 같은 출력(테스트 재현성). random 사용 금지.
- 한국어 조사는 _josa/_J/_ro 헬퍼 사용 — "이(가)" 식 병기 노출 금지.
- LLM 작성은 챕터 단위 '근거 슬롯 기반 작성'(해석 챕터) + 질문 분류 + 재윤문 (docs/06, 절대규칙15 개정 2026-06-11). 전체 PDF 원샷 freeform 금지·계산 LLM 위임 금지. 각 챕터는 룰 골격+검증 사실 슬롯만 근거, 근거 밖 사실(간지·별·연도·수치)은 factcheck 하드 차단. 프롬프트 변경 시 가드 6케이스 clean 재확인.
