# Instagram Story 템플릿

> 역할: 온기(warm) 오디언스 활성화 → 카톡/프로필 링크 전환.
> 스티커(투표/질문)로 참여 유발 → 답글·DM. 보장표현 금지.

## 3프레임 기본 세트

### 프레임 1 — 후킹 + 투표 스티커
```
"연애만 시작하면 식는다?"
일지에 답이 있을 수 있어요.

[투표 스티커]  나도 그래 / 난 반대야
```

### 프레임 2 — 미니 인사이트
```
일지가 '나의 무의식 본능' 자리예요.
여기가 흔들리면 시작은 빠른데 유지가 어렵죠.

(자세한 건 다음 카드에서)
```

### 프레임 3 — 질문 스티커 + CTA
```
당신 일지가 궁금하다면?

[질문 스티커]  생년월일 남겨줘 → 한 줄 봐줄게
프로필 링크에서 미니 리포트 신청 가능 🔗
```

## 변형 세트 (오퍼 푸시용)
- 프레임1: "이번 주 한정 미니 리포트" + 카운트다운 스티커(운영 안내, 공포 X)
- 프레임2: 무엇을 받는지(연애/직업/재물 1택, 참고용 해석)
- 프레임3: 링크 스티커 → 구글폼

## 운용 규칙
- 하루 3프레임 1세트 최소. 투표·질문 스티커 매 세트 1개 이상.
- DM 응답을 코어 상담 전환 기회로 사용.
- 외부링크는 링크 스티커/프로필 링크로(본문 링크 페널티 회피).
