# 상담 CTA 뱅크 (보장표현 無)

> 3종 강도(소프트/직접/마감임박)로 채널·상황에 맞게 선택.
> 금지: 결과·시기 보장, 공포 조장, 자격증 명시. "참고용 해석" 안전장치.

## 소프트 (도달 콘텐츠 끝 — 거부감 최소)
- "당신 일주는 어떤가요? 댓글에 적어주시면 한 줄 봐드릴게요."
- "본인 얘기 같다면, 프로필 링크에서 내 명식 가볍게 확인해 보세요."
- "더 궁금하면 채팅 주세요. 방향만 짚어드릴게요."

## 직접 (오퍼 콘텐츠 — 전환 목적)
- "이번 주 미니 사주 리포트: 가장 궁금한 1가지를 명식 근거와 함께 정리해 드려요. 신청은 프로필 링크."
- "연애·직업·재물 중 1택 — 참고용 해석 리포트로 기준 하나 가져가세요."
- "1:1 맞춤 상담은 채팅으로 신청 가능해요. 고민 1개를 구체적으로 풀어드립니다."

## 마감 임박 (운영 안내 — 시간 보장 아님)
- "이번 주 미니 리포트 신청은 오늘까지 받아요. (다음 주는 상담 위주 운영 예정)"
- "한정 인원이라 마감되면 다음 회차로 안내드려요."

## 결제 안내 문구 (송금/계좌 — 신뢰 보강)
```
신청 절차:
1. 구글폼에 생년월일시·성별·고민 1개 작성
2. 안내드린 계좌로 입금 (카카오 송금 가능)
3. 입금 확인 후 리포트/상담 일정 안내

* 단순 변심 환불 정책: [예: 작업 착수 전 전액 환불]
* 참고용 해석 자료이며 결과를 보장하지 않습니다.
```

## 금지 예시 (대조)
- ✗ "이거 안 하면 올해 재물운 놓칩니다" (공포)
- ✗ "한 달 안에 정리됩니다" (시기 보장)
- ✗ "100% 적중 / 반드시" (결과 보장)
- ✗ "명리상담사 1급 직접" (자격 명시)
