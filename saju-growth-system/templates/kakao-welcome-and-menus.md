# 카카오 채널 웰컴 + 채팅방 메뉴 (Day2 · Route D 미니랜딩)

> 브랜드 **사주도령**. 적용은 사용자가 관리자센터에서. 결과·시기 보장/공포 표현 금지(표시광고법).
> 적용 렌즈: L2 Brunson(Hook→Story→Offer·Attractive Character) + §A 심리기법.
> 카카오 글자 한도: 웰컴 내용 300자 · 자동응답 응답 400자 · 링크버튼 8자 · 소식버튼 10자 · 사용자 발화 14자.

## 0. 먼저 고칠 "고장" 3가지 (관리자센터 토글)
1. **채팅방 > 리스트 메뉴 사용 → ON** (현재 OFF라 메뉴가 안 뜸).
2. **자동응답 "💼직업/사업/이동 패키지" → 재연결** (현재 "연결되지 않음").
3. **웰컴 버튼 2개로** (현재 버튼1만, 버튼2 없음).

---

## 1. 웰컴메시지 (내용 — 300자 한도, 따뜻한 톤 유지 + 자기진단 유도)

```
안녕하세요, 사주도령입니다.
이곳을 찾아주신 것도 하나의 소중한 인연이라 생각해요.

저는 명리로 인생의 '큰 구조'를, 자미두수로 '세부 타이밍'을 교차해서 봅니다.
운명을 단정하는 게 아니라, 지금의 나를 더 깊이 이해하고
선택의 기준을 함께 찾는 일이에요.

그동안 많은 분들의 고민을 함께 봐 왔습니다.

먼저, 지금 가장 궁금한 한 가지부터 골라볼까요?
아래 메뉴에서 [나에게 맞는 풀이 고르기]를 눌러주세요 ↓
```
약 250자(300 한도 내). 이미지: 현재 四柱命理學 이미지 유지 + (가능하면) "풀이 구성 한 장" 이미지 추가.
[심리기법: 호감(따뜻한 톤)·권위(명리×자미 교차)·일관성(작은 클릭 유도)] [윤리가드: "단정하지 않는다" 명시, 보장표현 0]

**웰컴 버튼(2개):**
- 버튼1 (소식): `구성·가격 보기` → '사주풀이 구성' 또는 '가격' 소식 연결
- 버튼2 (소식): `실제 후기 보기` → 후기 소식 연결
[심리기법: 사회적증거(후기)·처리유창성(다음 행동 명확)]

> 사회적증거 강화 옵션: 첫 문단에 "그동안 7만 건 넘는 상담을 함께해 왔어요" 사용 가능(현재 IG/Threads 바이오의 '7만+건' 주장과 일치 — **사실일 때만** 사용).

---

## 2. 채팅방 리스트 메뉴 — 순서 (앵커링·일관성)

1. ☯ 나에게 맞는 풀이 고르기  ← 자기진단(첫 클릭)
2. 📖 사주풀이 구성 보기      ← 권위(무엇을 받나)
3. 💰 가격 보기              ← 프리미엄→코어→입문 앵커링
4. 💗 실제 후기 보기          ← 사회적증거(가격 직후 저항 완충)
5. ✅ 신청 방법 (필수)        ← 결제 직전
6. 💑 연애·결혼 흐름 보기
7. 💼 직업·사업·이동 결정 기준  ← (복구)

> 배치 원리: 가격=구성 직후 / 후기=가격 직후 / 신청=후기 직후.

---

## 3. 각 메뉴 자동응답 (400자 한도, 붙여넣기용)

### ① 나에게 맞는 풀이 고르기 (자기진단 — 일관성/약속)
```
지금 가장 마음이 가는 주제 하나를 골라보세요.

▶ 연애·결혼의 흐름과 타이밍이 궁금하다
   → 메뉴 [연애·결혼 흐름 보기]
▶ 이직·사업·이동, 지금 움직여도 될지 궁금하다
   → 메뉴 [직업·사업·이동 결정 기준]
▶ 내 사주의 큰 구조부터 알고 싶다
   → 메뉴 [사주풀이 구성 보기]

고른 주제에 맞춰 명리(큰 구조)와 자미두수(세부 타이밍)를 교차해 해석해 드려요.
바로 신청은 [신청 방법(필수)]에서 안내합니다.
```
[심리기법: 선택단순화·일관성(작은 예스)·자기진단] [윤리가드: 보장표현 0]

### ② 사주풀이 구성 보기 (권위 — 정통 12항목 유지)
```
[정통 사주 풀이 — 무엇을 보는가]
1) 사주 원국  2) 오행의 상호작용  3) 십성 분석
4) 초년·중년·말년운 및 종운  5) 신년운세  6) 용신
7) 신강·신약  8) 조심할 부분·개운 방향  9) 신살
10) 합·형·충·파·해 원진  11) 신살과 길성  12) 운세 흐름

여기에 자미두수 교차로 '세부 타이밍'을 더해 봅니다.
→ 가격은 [가격 보기], 신청은 [신청 방법(필수)]
```
[심리기법: 권위·구체성·처리유창성]

### ③ 가격 보기 (앵커링·디코이 — 프리미엄 먼저, 코어 강조)
```
[사주도령 풀이 안내]

■ 심층 교차(프리미엄) · 59,000원
  명리 × 자미두수 12궁 교차 + 향후 5년 흐름 + 추가질문 1회

■ 교차 리딩(코어) · 29,000원  ★ 가장 많이 선택해요
  원국 · 2026 대운 · 궁금한 1주제 심화 + 자미 교차 + 추가질문 1회

■ 미니 리딩(입문) · 9,900원
  연애·직업·재물 중 1가지 핵심 요약

옵션) 궁합 추가 +20,000 · 당일 우선납기 +5,000
[재오픈 기념가: ○월○일까지]   ← 사용자 확정(기간·이유 사실대로)

신청은 [신청 방법(필수)]에서.
```
[심리기법: 앵커링(프리미엄 먼저)·디코이(코어 ★)·진짜희소성(기간한정)] [윤리가드: 상시할인 아님·마감 사실일 때만]

### ④ 실제 후기 보기 (사회적증거 — 기존 후기 소식 연결)
```
사주도령을 찾아주신 분들의 실제 후기예요. (동의받은 후기만 게시)
"단순한 몇 줄이 아니라, 제 상황을 깊이 짚어주셨어요."

→ 후기 모음 보기: 아래 [후기 보러가기]
풀이가 궁금하면 [나에게 맞는 풀이 고르기]부터.
```
버튼(소식): `후기 보러가기`
[심리기법: 사회적증거·구체성(고객 언어)] [윤리가드: 동의받은 실후기만·과장 0]

### ⑤ 신청 방법 (필수) (Three Tens — 상품·사람·채널 확신 + 계좌이체 신뢰)
```
[신청 방법]
아래를 채팅으로 보내주세요.
▶ 이름 :
▶ 생년월일 (양력/음력) :
▶ 태어난 시간 (모르면 '모름') :
▶ 성별 :
▶ 가장 궁금한 1가지 :
▶ 찾아주신 경로 :

입금: [은행 · 계좌번호 · 예금주]   ← 사용자 확정
입금자명은 신청 성함과 같게 적어주시면 바로 확인돼요.
확인 즉시 '확인되었습니다'를 보내드리고, [납기: 예 24시간] 내 PDF를 전달합니다.
늦어지면 이 채팅으로 바로 문의 주세요 — 끝까지 책임지고 보내드려요.

* 결과를 보장하는 점이 아니라, 참고용 해석 자료입니다.
```
[심리기법: 리스크리버설·사회적증거(경로 질문)·Three Tens(상품·사람·채널)] [윤리가드: 절차·납기의 사실 약속만, 보장 0]
> 플레이스홀더 [예금주/계좌] [납기] 는 사용자 입력 필요(플랜 §승인입력 2).

### ⑥ 연애·결혼 흐름 보기 (기존 연애 패키지 정합 — 가격은 §가격표로 통일)
```
[연애·결혼 풀이 항목]
✅ 연애운 + 결혼운 — 다가올 사랑·결혼의 흐름
✅ 연애 + 결혼 + 재물 — 사랑과 돈이 함께 열리는 시기
✅ 상대와의 궁합 + 결혼운 — 두 사람의 인연과 결혼 후 흐름

명리(구조) × 자미두수(타이밍) 교차로 '선택의 기준'을 함께 봐요.
→ 가격은 [가격 보기], 신청은 [신청 방법(필수)]
```
[심리기법: 구체성·권위] [윤리가드: "선명하게 보여드린다"류 단정 → "흐름을 함께 본다"로 순화]
> 정정: 기존 메뉴의 개별가(1.8/2.7/5/5.5만)는 §가격표 3단으로 통일(궁합은 +20,000 옵션).

### ⑦ 직업·사업·이동 결정 기준 (복구 — 죽은 메뉴 내용 채움)
```
[직업·사업·이동 풀이 항목]
✅ 직업운 — 나에게 맞는 일의 방향·강점
✅ 직업 + 재물 — 일과 돈의 흐름을 함께
✅ 이동·이사 — 지금 움직여도 될지, 방향·시기 참고
✅ 사업 적합도 — 조직형 / 독립형 성향 해석

'결정의 기준'을 명리·자미두수 교차로 짚어드려요(단정 아님).
→ 가격은 [가격 보기], 신청은 [신청 방법(필수)]
```
[심리기법: 선택단순화·권위] [윤리가드: "결정 기준 참고", 결과 단정 금지]
