# 카카오채널 메시지 템플릿

> 발송 전 필독: `research/kakao-compliance-notes.md`
> (광고) 메시지 3필수: (광고) 표시 + 전송자 연락처 + 무료 수신거부. 21~08시 금지. 친구당 1통/24h.
> 보장·시기 단정 금지. 발송 전 단가 확인.

---

## 1. 무료 정보성 게시 (광고 아님 — 홈/포스트)

> 판매 유도 없이 가치만. 채널 피드 노출용.

```
[오늘의 사주 한 줄]
재물운이 좋다는 일주인데 늘 돈이 모이지 않는다면,
'버는 힘'과 '지키는 힘'이 명식에서 따로 노는 경우가 많아요.

당신 일주는 어느 쪽인지 궁금하다면 댓글이나 채팅으로 알려주세요.
```

---

## 2. (광고) 브랜드 메시지 — 입문 오퍼 (D2 발송)

> 광고성으로 분류하면 (광고)·수신거부는 카카오가 자동 처리. 아래는 본문.
> 연락처 라인은 반드시 포함.

```
(광고) 서담선생

막연한 불안 대신, '내 사주를 보는 기준' 하나를 가져가 보세요.

이번 주 한정 미니 사주 리포트
- 연애 / 직업 / 재물 중 가장 궁금한 1가지
- 당신 명식 근거와 함께 한 장으로 정리해 드립니다 (참고용 해석)

신청: [구글폼 링크]
문의: [카카오톡 채팅 / 연락처]

* 결과를 보장하는 서비스가 아니며, 참고용 해석 자료입니다.
무료 수신거부: [카카오 자동 안내]
```

체크: (광고) 표기 / 연락처 / 무료 수신거부 / 보장표현 0건 / 발송 08~21시 / 24h 내 중복 없음.

---

## 3. 정보성 후속 (D4 — 비광고, 사례 해석)

```
[해석 과정 공개]
지난번 '재물이 새는 패턴' 이야기에 문의가 많았어요.

실제로 본 익명 사례(동의받음)에서는,
일지와 시지의 관계가 핵심이었습니다.

본인 명식이 궁금하다면 채팅으로 생년월일시를 남겨주세요.
가볍게 방향만 짚어드릴게요.
```

---

## 4. 마감 임박 (D6 — 시간 보장 표현 금지)

```
(광고) 서담선생

이번 주 미니 리포트 신청은 오늘까지 받아요.
(다음 주에는 코어 상담 위주로 운영할 예정이에요)

신청: [구글폼 링크]
문의: [연락처]

* 참고용 해석 자료입니다.
무료 수신거부: [카카오 자동 안내]
```

> "오늘까지 받아요"(운영 안내)는 OK. "오늘 안 하면 운이 나빠진다"류 공포·보장은 금지.
