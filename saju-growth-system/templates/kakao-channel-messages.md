# 카카오 채널 5일 재활성 캠페인 (Day3 산출물 · Soap Opera Sequence)

> 브랜드 **사주도령**. 매출 엔진 = 친구 ~4,000 재활성(트랙A). 적용은 사용자가 관리자센터에서.
> 적용 렌즈: L2 Brunson(Hook-Story-Offer·Soap Opera 5비트) · L3 Kennedy(이유있는마감) · L5 Cardone(다회 접촉) · §A 심리기법.
> **규정:** 광고성 메시지는 (광고)+전송자 연락처+무료 수신거부(카카오 자동) / **발송 08:00~20:55**(야간 20:55~08:00 차단) / 친구당 1통/24h / 결과·시기 보장·공포 표현 금지(표시광고법 §3·§8).

## 발송 구성 & 비용 (기본: 유료 3 + 무료 소식 2)

|---|---|---|---|---|
| D1 | M1 가치+재오픈 예고 | 유료 채널 메시지 | 오전 9시 또는 저녁 7시 | 6~8만원 |
| D2 | M2 스토리/오픈루프 | 무료 소식(피드) | 저녁 7시 | 0 |
| D3 | **M3 오퍼 공개(핵심)** | 유료 채널 메시지 | 오전 9시 | 6~8만원 |

## M1 (D1) — 가치 선행 + 재오픈 예고 (상호성·호감·오픈루프)
형태: 유료 채널 메시지(이미지형 권장) · 약 300자
```
(광고) 사주도령

그동안 채널을 비워뒀는데, 다시 문을 엽니다.

돌아온 김에 작은 것부터 드릴게요.
[일주로 보는 2026 키워드] — 내 사주의 '큰 흐름' 한 가지를
이번 주에 무료로 짚어드립니다. 채팅으로 생년월일시만 남겨주세요.

그리고 이번 재오픈에 맞춰, 5일간만 신청을 받아요.
교차 리딩을 다시 엽니다. 자세한 건 모레 안내드릴게요.

문의: 이 채팅 / 무료 수신거부: [카카오 자동]
```
[심리기법: 상호성(무료 선제공)·호감·자이가르닉(오픈루프 '모레 안내')] [윤리가드: 보장표현 0·무료는 진짜 제공]

## M2 (D2) — 스토리 / 오픈루프 (Attractive Character·사회적증거)
형태: 무료 소식(피드) · 약 300자
```
[사주도령 이야기]
처음엔 제 사주 하나 제대로 못 읽어서, 답답해서 공부를 시작했어요.

명리만 보던 시절엔 '큰 구조'는 보이는데 '언제'가 흐릿했습니다.
자미두수를 교차하면서부터, 흐름에 '타이밍'이 얹히더라고요.

그동안 많은 분들과 그 '때'를 같이 읽어 왔습니다.
내 사주의 큰 구조가 궁금하면 [나에게 맞는 풀이 고르기]부터 눌러보세요.
```
[심리기법: 스토리텔링(백스토리·결함)·권위·일관성] [윤리가드: 단정 없이 '함께 읽는다' 프레이밍]

## M3 (D3) — 오퍼 공개 = 핵심 (에피파니·앵커링·디코이·진짜희소성)
형태: 유료 채널 메시지(이미지형 권장) · 약 350자
```
(광고) 사주도령

재오픈 첫 주, 5일간만 신청을 받아요. [○월○일 ~ ○월○일]

■ 심층 교차(프리미엄) 59,000원
  명리 × 자미두수 12궁 교차 + 향후 5년 흐름 + 추가질문 1회
■ 교차 리딩(코어) 29,000원  ★ 가장 많이 선택해요
  원국·2026 대운·궁금한 1주제 + 자미 교차 + 추가질문 1회
■ 미니 리딩(입문) 9,900원

명리로 큰 구조를, 자미두수로 세부 타이밍을 교차해 봅니다.
운명을 단정하는 게 아니라, 선택의 기준을 함께 찾는 풀이예요.

이번 기간이 지나면 다음 배치까지 접수를 닫아요.
신청: 이 채팅에 "재오픈"과 함께 생년월일시를 남겨주세요.
문의: 이 채팅 / 무료 수신거부: [카카오 자동]
```
[심리기법: 앵커링(프리미엄 먼저)·디코이(코어 ★)·진짜희소성(5일)·구체성] [윤리가드: 상시할인 아님·보장 0]

## M4 (D4) — 히든 베니핏 + 후기 (사회적증거·이의 해소)
형태: 무료 소식(피드) · 약 300자
```
[자주 받는 질문]
"사주는 다 비슷한 말 아닌가요?"

명리만 보면 큰 틀은 같아 보일 수 있어요.
그래서 저는 자미두수를 교차해 '내 상황의 세부 타이밍'까지 봅니다.
같은 일주라도 결론이 달라지는 이유예요.

실제 후기는 [실제 후기 보기]에서 확인하실 수 있어요.
(동의받은 후기만 게시합니다)
재오픈 첫 주 접수는 모레까지예요.
```
[심리기법: 사회적증거·권위(차별점)·이의처리(루핑)·진짜희소성] [윤리가드: 후기 동의·과장 0]

## M5 (D5) — 마감 D-day (이유 있는 마감 · 손실회피)
형태: 유료 채널 메시지 · 약 300자
```
(광고) 사주도령

재오픈 첫 주 접수는 오늘까지예요. [○월○일]

한 분 한 분 명식을 직접 검수해 써 드리느라
접수를 기간으로 나눠 운영해요. 내일부터는 접수를 잠시 닫습니다.
(지금 기간에 신청하시면 첫 배치로 받아보실 수 있어요)

망설여진다면, 지금 가장 궁금한 한 가지부터 가볍게 물어보셔도 돼요.
신청: 이 채팅에 "재오픈"과 함께 생년월일시 / 문의: 이 채팅
무료 수신거부: [카카오 자동]
```
[심리기법: 이유있는마감(한정 접수)·손실회피·진입장벽 낮추기] [윤리가드: 가격은 정가·할인 주장 아님. 기간 종료 시 실제로 접수를 닫을 것(아니면 가짜 희소성)]
