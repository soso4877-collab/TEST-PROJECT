## 왜 '소식 랜딩'인가 (Route B 근거)
- 웜(기존 카카오 친구) = 채팅 직행(Route D). 콜드(IG·Threads 신규) = **처음 보는 브랜드**라 바로 채팅에 생년월일시를 주기엔 신뢰가 부족.
- 그래서 콜드는 한 번 '소식'에서 ① 누구인지(차별점) ② 무엇을 받는지(구성) ③ 얼마인지(가격) ④ 믿을 수 있는지(후기·계좌 신뢰)를 **선이해**한 뒤 채팅으로 — 클릭 1회 더 들지만 콜드 전환율↑.
- IG 동선: 프로필 링크/게시물 CTA → 이 소식 → 채팅. (웜은 day4·웰컴 동선 그대로)

---

## 소식 랜딩 본문 (그대로 게시 · 카카오 채널 소식)

```
사주를 봐도, 내 얘기 같지 않으셨나요?

솔직히 저도 처음엔 제 사주조차 제대로 못 읽었어요.
명리로 큰 그림은 보이는데 '그래서 지금 뭘 해야 하지'에서 막혔거든요.
자미두수를 교차하면서부터 '언제'가 같이 보이기 시작했어요.

저는 사주도령입니다.
명리로 인생의 큰 구조를, 자미두수로 세부 타이밍을 교차해서 봐요.
운명을 단정하는 게 아니라, 지금의 나를 더 깊이 이해하고
선택의 기준을 함께 찾는 일이에요.


■ 무엇을 보나요 (정통 풀이 + 자미 교차)

사주 원국 · 오행의 상호작용 · 십성 분석
초년·중년·말년운 및 종운 · 신년운세 · 용신 · 신강신약
조심할 부분과 개운 방향 · 신살 · 합충형파해 · 운세 흐름

여기에 자미두수 12궁 교차로 '세부 타이밍'을 더합니다.
같은 일주라도 결론이 달라지는 이유예요.


■ 풀이 안내

심층 교차(프리미엄) · 59,000원
  명리 × 자미두수 12궁 교차 + 향후 5년 흐름 + 추가질문 1회
  (구성 상당 110,000 → 59,000)

교차 리딩(코어) · 29,000원   ★ 가장 많이 선택해요
  원국 · 2026 대운 · 궁금한 1주제 심화 + 자미 교차 + 추가질문 1회
  (구성 상당 55,000 → 29,000)

미니 리딩(입문) · 9,900원
  연애 · 직업 · 재물 중 1가지 핵심 요약
  사주가 처음이거나 한 번만 써보고 정하고 싶다면 여기서 시작하세요.
  가볍게 확인하고 다음 단계를 정하셔도 돼요.


■ 신청 전에 확인하세요

· 받으시는 것: 명리 × 자미두수 교차 해석 자료 + 추가질문 1회(코어·프리미엄)
· 입금 계좌는 신청 정보 확인 후 채팅으로 안내드려요 (예금주: 사주도령).
  입금자명을 신청 성함과 같게 적어주시면 바로 확인돼요.
· 받는 즉시 명식을 직접 검수해 작성하고, 입금 확인 후 24시간 이내
  (영업시간 기준)에 보내드려요. 대부분 더 빨리 도착해요.
· 전달이 늦어지면 채팅으로 바로 연락 주세요. 확인 즉시 직접 챙겨드려요.

* 결과를 보장하는 점이 아니라, 참고용 해석 자료예요.


■ 신청 방법

아래 [채팅으로 신청하기]를 누르고, "소식"과 함께 보내주세요.
이름 / 생년월일(양력·음력) / 태어난 시간(모르면 '모름') / 성별 / 가장 궁금한 1가지

잘 모르는 항목은 '모름'으로 보내셔도 돼요. 채팅으로 바로 여쭤볼게요.
먼저 [실제 후기]와 [풀이 구성]을 보고 결정하셔도 괜찮아요.
```

> 소식 글자 한도는 넉넉(수천 자) — 위 본문은 약 700자 내외로 콜드 가독성 우선.

## 후기 블록 (소식 하단 또는 별도 후기 소식 — 동의받은 것만)
```
실제로 받아보신 분들의 이야기예요. (동의받은 후기만 게시합니다)

"단순히 좋은 말이 아니라, 제 상황을 콕 짚어주셔서 정리가 됐어요."
"명리랑 자미두수를 같이 보니 '왜'가 이해됐어요."

— [여기에 실제 후기 2~3개를 캡처/인용으로 교체. 미보유 시 이 블록 생략]
```
> 주의: 후기는 **동의받은 실제 후기**만 사용. 창작·과장 금지(표시광고법). 미보유면 블록을 빼고 게시.

## IG → 소식 유도 동선 (콜드 진입)
- IG 게시물 CTA(전환글): "더 궁금하면 프로필 링크 → '사주도령 풀이 안내' 소식부터 보세요."
- 프로필 링크는 카카오 채팅 직링크가 기본(웜). 콜드 캠페인 기간엔 **소식 URL을 프로필 링크로 임시 교체 A/B** 가능 — 어느 쪽 전환율이 높은지 측정해 승자 고정.
- 측정: "소식" 키워드 신청 수 / 소식 조회 대비 채팅 전환(수동 시트, `tracking/daily-tracker.xlsx`).
