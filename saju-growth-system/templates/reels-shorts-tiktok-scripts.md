# 릴스 / 쇼츠 / 틱톡 스크립트 템플릿

> 도달 레버: 완시청률 + 공유. 첫 3초 후킹 필수. 원본성(워터마크/재탕 금지).
> AI 현실적 합성시 라벨. 보장표현 금지.
> 제작: 카드뉴스 우선 → 승자만 영상화. CapCut 템플릿(공식 배치/업로드 API 없음) 또는 인스타 릴스툴.

## 30초 릴스 구조 (자막 중심)

| 구간 | 시간 | 내용 | 자막 예시 |
|---|---|---|---|
| 후킹 | 0-3초 | 반증 질문 | "재물운 좋은 일주인데 왜 돈이 없을까?" |
| 전환 | 3-8초 | 호기심 갭 확장 | "이유는 명식의 '버는 힘 vs 지키는 힘'" |
| 본문 | 8-22초 | 핵심 패턴 2~3개 | "1. 식상 과다 / 2. 재성 무근 / 3. 비겁 쟁재" |
| 마무리 | 22-27초 | 자기관련성 | "당신은 몇 번에 해당될까요?" |
| CTA | 27-30초 | 소프트 전환 | "프로필 링크에서 내 명식 확인" |

## YouTube Shorts (검색 최적화)
- **제목**: 키워드 앞배치 — "2026 재물운 | 돈 새는 일주 3가지 #사주 #운세"
- 설명: 핵심 키워드 2~3줄 + 고정댓글에 링크.
- 자막: 9:16 중앙, 큰 글씨. 첫 3초 후킹 자막 고정.
- 끝에 "다음 편: 지키는 힘 키우는 법"(오픈루프 → 백카탈로그).

## TikTok (트렌드 친화)
- 후킹을 트렌드 사운드 비트에 맞춤.
- 해시태그 3개: #사주 #운세 #일주 (니치 + 광범위 혼합).
- 보장·운세 단정 회피("~일 수 있어요"). 엔터/성찰 톤(FYF 적격 유지).
- 실존인물 likeness 사용 금지.

## 제작 워크플로 (시간 최소)
1. `content/hooks-library.md`에서 후킹 선택 → 카드뉴스 먼저 제작.
2. 카드뉴스 성과 상위 1~2개만 영상화:
   - 옵션 A: CapCut 템플릿에 자막/이미지 교체 → 배치 export.
   - 옵션 B: 카드뉴스 슬라이드를 인스타 릴스 편집툴로 영상화.
3. Metricool로 IG/TikTok/Shorts 동시 예약발행(YT 썸네일은 수동).

## 자막 작성 규칙
- 한 화면 1~2줄. 첫 줄은 항상 후킹.
- 숫자·대조·질문으로 시각적 호기심.
- 보장표현·자격증·형식암시(PDF/책자) 톤 금지.
