# 플랫폼 정책 노트 — 도달·원본성·AI 라벨 (실행 요약)

> 검증일 2026-06-08. 근거 전문: `verified-source-map.md`

## 공통 진실 (4개 영상 플랫폼 수렴)

도달을 결정하는 **두 레버**:
1. **완시청률(watch-through / completion)** — 짧은 루프, 첫 3초 강한 후킹.
2. **의미있는 공유(meaningful share)** — "친구에게 보내고 싶은" 콘텐츠 + 명시적 공유 유도.

→ 사주 콘텐츠 설계 원칙: 빠른 페이오프 + "당신/내 일주는?" 같은 자기관련성 → 공유·답글 유발.

## 채널별 핵심

### Instagram
- **Reels**: <3초 이탈 패널티, >95% 시청 보상 + 댓글·리셰어·외부공유·팔로우 확률.
- **Explore**: 비팔로워 콜드도달 주력면. 낯선 이의 **저장·공유** 최적화.
- **원본성(중요 리스크)**: 테두리·워터마크·속도변경·크레딧은 원본 아님. 비원본 위주면 비팔로워 추천 제한.
  - **금지**: 틱톡 export(워터마크) 클립 재업로드, 타계정 캡처 재탕.
  - **필수**: 자기 내레이션·고유 프레이밍·관점 추가. 최근 30일 원본 다수면 추천 회복.
- **비즈니스 계정** 필요(스케줄러 자동발행·콘텐츠퍼블리싱 API 전제).
- 월 1회 **계정 상태**(설정>계정 상태) 점검.

### Threads
- 사려깊은 답글·체류시간 > 수동 조회. 대화유발 우대.
- → 질문형 후킹 + 오픈루프 + 게시 후 30분 자기답글(기존 `output/02_account_diagnosis.md` 레버와 일치).
- 본문 외부링크 페널티 가능성 → 링크는 **프로필/댓글**로.

### YouTube Shorts
- 소규모 테스트 → 유지율·만족도로 확장. **Engaged views**(완시청)가 진짜 신호.
- 과거 숏츠 재부상 가능 → 백카탈로그 유지. 키워드 제목("2026 운세", 일주명)으로 검색 노출.

### TikTok
- 완시청·공유·댓글 중심. **팔로워수·과거히트 직접요인 아님** → 콜드도달 가능.
- 트렌드 사운드 + 니치 해시태그. 보장 표현시 FYF(For You) 제외 위험.

## AI 생성 콘텐츠 라벨 (3사)

| 플랫폼 | 라벨 의무 | 면제 | 도달 영향 |
|---|---|---|---|
| Meta(IG) | 오인 가능 현실적 AI 이미지/영상/오디오 → "AI Info" | 일러스트/스타일 아트, AI 텍스트 | 원본·비기만이면 중립 |
| YouTube | 현실적 변형/합성 → Studio 토글 | 제작보조(스크립트/편집/색보정) | YPP는 "유의미 독창성" 요구(대량 저품질 표적) |
| TikTok | 완전 AI/중대 변형 현실적 → 라벨 | 경미 편집 | **라벨이 비위반 콘텐츠 도달 감소 안 시킴(명시)** |

**실행 규칙**: 사주 카드뉴스/일러스트는 대체로 면제. 포토리얼 AI 인물/장면은 라벨. 실존인물 likeness 금지.

## 한 장 요약 (도달 체크)
- [ ] 첫 3초 후킹 있는가
- [ ] 완시청 가능한 짧은 길이/루프인가
- [ ] 공유·답글 유발 장치(자기관련 질문)인가
- [ ] 원본인가(워터마크/재탕 아님)
- [ ] (현실적 AI면) 라벨 토글 했는가
