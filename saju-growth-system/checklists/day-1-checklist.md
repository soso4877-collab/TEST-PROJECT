# D1 체크리스트 — 퍼널 구축 (가장 중요한 하루)

> 목표: 내일(D2) 첫 매출이 가능하도록 결제·주문·배송·콘텐츠·측정 인프라를 1일 안에 세운다.

## 1. 오퍼 확정 (30분)
- [ ] 입문 미니 리포트 가격 확정(권장 29,000원)
- [ ] 코어 1:1 상담 가격 확정(권장 59,000원)
- [ ] 각 오퍼가 무엇을 주는지 1줄 정의(연애/직업/재물 1택, 참고용 해석)

## 2. 결제·주문 플로 (40분)
- [ ] 구글폼 주문서 생성: 생년월일시·성별·고민 1개·연락처
- [ ] 입금 계좌/카카오 송금 안내 문구 작성(`templates/consultation-cta-bank.md`)
- [ ] 환불 정책 1줄(예: 작업 착수 전 전액 환불) + "결과 보장 안 함" 안전문구
- [ ] 정산 시트 생성(주문→입금→배송 상태)

## 3. 입문상품 배송 점검 (40분)
- [ ] `sajugen/`으로 미니 리포트 1건 테스트 생성(PDF 산출 확인)
- [ ] 본인 LLM 보조 해석 코멘트 1개 작성·검수(보장표현 0건)
- [ ] 카톡 전달 형식 확정(PDF + 코멘트)

## 4. 콘텐츠·측정 인프라 (60분)
- [ ] Google Sheets에 `content/7-day-content-calendar.csv` 복제(콘텐츠 DB)
- [ ] Metricool 계정 생성 + IG/Threads/TikTok/YT 연결(IG는 비즈니스 계정 확인)
- [ ] `tracking/metrics-schema.csv` 복제해 메트릭 시트 생성
- [ ] (옵션) `repurpose.py`로 오늘 후킹 1개 → 드래프트 생성 테스트

## 5. 첫 게시 + 예고 (40분)
- [ ] Threads 3개 게시(반증형 + 오픈루프), 게시 후 30분 자기답글
- [ ] IG 카드뉴스 1개(재물 패턴 3가지)
- [ ] 카톡 무료 조직게시: "내일 한정 미니 리포트 예고"(광고 아님)

## 6. 마감 점검
- [ ] `checklists/compliance-checklist.md` 1회 통과
- [ ] D2 (광고) 브랜드 메시지 초안 준비 + 관리자센터 실단가 확인
