## 채널 우선순위 (7일)
1. **카카오채널** — 현금화 1순위(소유·옵트인). 가장 빠른 매출.
2. **Threads + Instagram** — 도달·리드. 기존 강점 자산(19K/24K).
3. **YouTube Shorts / TikTok** — 재활용 위주, 시간 최소 투입. 콜드도달 보너스.

## "가장 승산 있는 게임" 운영 원칙
- **카드뉴스 우선 제작**(텍스트 강점 활용) → 성과 상위만 릴스/숏츠로 전환(영상 시간 최소).
- D3에 포맷 A/B(카드뉴스 vs 자막릴스) + 후킹 3종 테스트 → D4부터 승자 더블다운.
- 영상은 CapCut 템플릿(공식 배치/업로드 API 없음) 또는 카드뉴스→인스타 릴스툴로 깔끔하게.

## 링크 전략 (퍼널 단축)
- 모든 채널 종착 = **카카오채널/오픈챗 → 구글폼 주문 → 송금/계좌 결제**.
- Threads/IG 본문 링크 회피 → 프로필 단일 링크(링크트리 또는 카톡 단일).
- 카톡 메시지에만 직접 주문 링크 허용(단, (광고) 컴플라이언스).
