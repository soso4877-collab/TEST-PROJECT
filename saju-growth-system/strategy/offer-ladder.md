# 오퍼 사다리 (Offer Ladder) — 사주도령

> 브랜드 **"사주도령"(@bachelor.saju)** 만 사용. (서담선생은 당근 별개 프로젝트 — 혼동 금지)
> 모든 카피 **결과·시기 보장 표현 금지**(표시광고법 §3·§8). "참고용 해석/학문적" 프레이밍. 자격증 명시 금지. → `checklists/compliance-checklist.md`
> 가격은 시장조사 기반(크몽 실판매 앵커 18,000~33,000). 적용 렌즈: L1 Hormozi(가치방정식·스태킹·리스크리버설) · L3 Kennedy(이유있는마감·USP·프리미엄).

## 3단 사다리 (시장조사 기반 가격 리셋)

| 오퍼 | 가격 | 타깃 | 욕구 트리거 | 명리+자미 차별 | 배송 | 7일 판매확률 |
|---|---:|---|---|---|---|---|
| **입문 미니 리딩** | **9,900** | 콜드/웜 팔로워·카톡친구 | "내 명식 핵심 1가지를 정확히" | 핵심 1축 요약 | sajugen 미니 PDF | 높음(콜드 깔때기) |
| **교차 리딩(코어·메인)** | **29,000** | 연애/직업/이동 중 1개 결정 임박 | "입체적 자기이해·선택 기준" | 명리 중심 + 자미 보조 | 풀구성 PDF + 추가질문 1회 | **높음 — 첫 판매(트랙A)** |
| **심층 교차(프리미엄)** | **59,000** | 고관여·결정 무게 큰 고객 | "전 영역 종합 + 5년 흐름 + 타이밍" | **명리×자미두수 12궁 교차** | 확장 PDF(20p+) + 추가질문 1회 | 중상(객단가 핵심) |
| (옵션) 궁합 추가 | +20,000 | 관계 고민 | 상대와의 흐름 | 두 명식 교차 | 추가 섹션 | — |
| (옵션) 당일 우선납기 | +5,000 | 다급한 수요 | 24h 내 수령 | — | 우선 처리 | — |

> 가격 근거: 코어 29,000 = 크몽 리뷰 1위 서비스 실판매 중앙값(18,000~33,000). 프리미엄 59,000 = 크몽 고가밴드(55,000~60,000) + 자미두수 전문성 프리미엄. 입문 9,900 = 1만원 미만 심리장벽·첫상담 관행가. 조정 시 §9 매출수학 동반 갱신.

## L1 가치 스태킹 — 코어/프리미엄 가격을 "싸 보이게"(앵커링·처리유창성)

코어 29,000 구성요소를 개별가로 분해해 합산가 대비 체감 가성비를 높인다(보장표현 아님, "상당" 표기).

```
[코어 교차 리딩 — 구성]
· 사주 원국 분석(성향·신강신약)      상당가 19,000
· 2026 대운·세운 흐름               상당가 15,000
· 가장 궁금한 1주제 심화 해석        상당가 12,000
· 자미두수 교차 포인트 + 핵심 조언    상당가  9,000
──────────────────────────────
합산 상당가 55,000  →  교차 리딩 29,000
```
```
[프리미엄 심층 — 구성]
· 코어 전체 포함                      상당가 55,000
· 자미두수 12궁 교차 분석             상당가 25,000
· 향후 5년 운세 흐름·시기 포인트       상당가 20,000
· 추가질문 1회 + 우선 응대            상당가 10,000
──────────────────────────────
합산 상당가 110,000  →  심층 교차 59,000
```
[심리기법: 앵커링·처리유창성·구체성] [윤리가드: "상당가"는 구성가치 안내일 뿐 허위 정가/상시할인 아님 — 표시광고법 통과]

## 즉시 판매할 첫 오퍼 — 트랙A(카카오 4,000 친구 재활성)

매출 엔진은 **카카오 친구 ~4,000명 재활성**. 코어 29,000(대중) + 프리미엄 59,000(객단가)을
**기간 한정 재오픈가**로 동시 투입. 입문 9,900은 콜드 SNS 신규를 데우는 깔때기(트랙B), 입문/코어 구매자는 다음 단계로 업셀.
가격 제시 순서: 프리미엄 먼저 → 코어를 "가장 인기" 기본 선택지로(디코이·앵커링).

근거(§9): 코어 29,000 단독이면 일 약 7건, 프리미엄 59,000은 약 3.4건. 권장 믹스 = 프리미엄 2 + 코어 3 ≈ 205,000원(5건). 재활성 피크일엔 초과 여지.

### 배송 플로 (코어/프리미엄)
1. 구글폼으로 생년월일시·성별·고민 1개(연애/직업/이동) 수집.
2. sajugen으로 명리+자미두수 교차 PDF 생성(`sajugen/` CLI/FastAPI, 작동 확인).
3. LLM 보조로 교차 해석·선택 기준 코멘트 작성 → **사람 검수**(명식 검수 포함).
4. 입금 확인 → 카톡으로 PDF + 코멘트 전달 → 추가질문 1회 응대.
- 처리시간 목표: 코어 25~40분 / 프리미엄 40~60분(템플릿·AI 보조로 단축). 용량 병목 아님 — 병목은 문의 수.

### 배송 플로 (입문, 콜드 깔때기)
1. 구글폼 수집 → 2. sajugen 미니 PDF → 3. 핵심 1축 코멘트 검수 → 4. 입금 후 전달.
- 처리시간 목표: 10~20분.

## L3 "이유 있는 마감" — 기간 한정 (상시할인 폐지 · Kennedy)

상시할인은 정가 신뢰를 깎고 표시광고법 리스크(상시할인=실질정가). 대신 **진짜 마감일 + 진짜 이유**를 명시한다.

**기간한정 카피(틀 — 날짜만 채움, 5일 확정):**
> "[시작일]~[종료일], 재오픈 첫 주 5일간만 신청을 받아요.
>  한 분 한 분 명식을 직접 검수해 정성껏 써 드리기 때문에,
>  접수를 기간으로 나눠 운영합니다. 이 기간이 지나면 접수를 잠시 닫아요.
>  지금 기간 안에 신청하시면 첫 배치로 받아보실 수 있어요."

[심리기법: 진짜 희소성(한정 접수기간)·일관성·구체성] [윤리가드: 가격은 정가 — '할인' 주장 아님(29,000은 시장조사 실가). 희소성은 '5일 접수'에만 둘 것. 기간 종료 시 실제로 접수를 닫아야 함(아니면 가짜 희소성)]

## 컴플라이언스 통과 카피 예시 (보장표현 無)

**OK (참고·해석·선택기준 프레이밍)**
- "당신 명식에서 재물이 어떻게 흐르는지 한 장으로 정리해 드립니다."
- "연애·직업·재물 중 가장 궁금한 1가지를, 명식 근거와 함께 해석해 드려요."
- "막연한 불안 대신, 내 사주를 보는 기준 하나를 가져가세요."
- "명리로 큰 구조를, 자미두수로 세부 타이밍을 — 두 체계를 교차해 봅니다."

**금지 (보장·단정·자격·공포)**
- ✗ "한 달 안에 정리됩니다 / 반드시 풀립니다 / 100% 적중 / 확실히 좋아집니다"
- ✗ "이거 안 보면 큰일 납니다"(공포조장)
- ✗ "심리상담사 1급이 직접"(자격 명시)

## 가격·마진 메모
- 입문 9,900: 처리 10~20분, 콜드 리드 확보·업셀 진입용.
- 코어 29,000: 처리 25~40분(AI 보조), 메인 볼륨·트랙A 첫 판매. 일 약 7건 = 20만원.
- 프리미엄 59,000: 처리 40~60분, 객단가 핵심. 일 약 3.4건 = 20만원.
- 권장 믹스: 프리미엄 2 + 코어 3 ≈ 205,000원.

## 계좌이체 신뢰 카피 (결제 직전 배치 — 보장표현 無)

> 계좌이체는 "입금 후 못 받을까" 불안이 최대 마찰점. 결제 직전에 5요소를 한 메시지로 묶는다(증거→절차→수령물→타임라인→문의처).
> [L3 Belfort Three Tens 적용: 상품·사람·채널 확신을 동시에]

- **증거 선노출(상품 확신):** "신청 전 [실제 후기]와 [사주풀이 구성]을 먼저 확인하실 수 있어요."
- **입금 절차(채널 확신):** "입금 계좌는 신청 정보 확인 후 채팅으로 안내드려요(예금주: 사주도령). 입금자명을 신청 성함과 같게 적어주시면 바로 확인됩니다. 확인 즉시 '확인되었습니다'를 보내드려요."
- **수령물 명시:** "받으시는 것: 명리×자미두수 교차 해석 PDF(풀구성) + 추가질문 1회 응대."
- **배송 타임라인(언더프로미스→오버딜리버):** "받는 즉시 명식을 직접 검수해 정성껏 작성하고, 입금 확인 후 24시간 이내(영업시간 기준)에 전달드려요. 대부분 그보다 빨리 도착합니다."
- **문의처·책임(사람 확신):** "전달이 지연되면 이 채팅으로 바로 문의 주세요. 끝까지 책임지고 보내드립니다."

[심리기법: 사회적증거·리스크리버설·운영투명성(노력 가시화)·언더프로미스→오버딜리버] [윤리가드: 절차·납기의 사실 약속만, "반드시/100%/결과보장" 금지]
(상호=사주도령 노출, 예금주명 표기 OK. 계좌번호는 공개 노출 없이 신청 후 1:1 안내.)

## 추가질문 정책 (범위·횟수·응답시간 — 컴플레인 예방 = L1 리스크리버설)

- **포함 범위:** 전달된 PDF 해석 이해를 돕는 질문 1회(코어/프리미엄). 예) "이 부분 타이밍을 제 상황에 맞게 다시 설명해 주세요."
- **제외 범위:** 새 명식 추가 풀이·무제한 상담·결과 단정 요구는 별도 오퍼(프리미엄/궁합)로 안내.
- **응답 시간:** 영업시간 기준 ○○시간 내 1회 답변(야간차단 20:55~08:00은 익일 오전).
- **AI 보조 + 사람 검수:** 답변 초안은 AI 보조, **반드시 사람 검수 후 발송**.
- **개인정보:** 생년월일시 등 민감정보는 sajugen 로컬 처리. 외부 3rd파티 LLM 불필요 업로드 금지(privacy-safe 워크플로는 §13 Later).
- **표현 가드:** 추가질문 답변에도 결과·시기 보장 금지. "참고용 해석·선택 기준" 유지.
