# 오퍼 사다리 (Offer Ladder)

> 모든 카피는 **결과·시기 보장 표현 금지**(표시광고법 §3). "참고용 해석/학문적" 프레이밍.
> 브랜드 "서담선생"만. 자격증 명시 금지. → `checklists/compliance-checklist.md`

## 3단 사다리 (승인 플랜 §8 정합 — 차별화 축 = 명리×자미두수 교차 PDF)

| 오퍼 | 가격 | 타깃 | 욕구 트리거 | 명리+자미 차별 | 배송 | 7일 판매확률 |
|---|---:|---|---|---|---|---|
| **미니 사주 리포트(입문)** | 19,000~29,000 | 기존 팔로워·카톡친구, 가벼운 호기심 | "내 명식 핵심 1가지를 정확히 짚어줌" | 핵심 구조 1축 요약 | sajugen PDF + 본인 LLM 보조 코멘트 | 높음(콜드 깔때기) |
| **교차 리딩(코어·메인)** | **59,000** | 연애/직업/이동 중 1개 결정 임박 | "입체적 자기이해·선택 기준" | **명리(큰 구조)×자미두수(세부 타이밍) 교차** | 풀구성 PDF + 카톡 추가질문 1회 | **높음 — 첫 판매(트랙A)** |
| **종합 패키지(업셀)** | 149,000+ | 코어 만족·고관여 고객 | "전 영역 종합 + 시기 흐름 + 후속" | 교차리딩 + 사건성 + 후속 상담 | 확장 PDF + 심화 상담(선택: 통화) | 중(후순위·붙으면 가속) |

> 코어 가격 59,000은 승인 플랜 권장값. 조정 시 §9 매출 수학(일 약 3.4건)도 함께 갱신.

## 즉시 판매할 첫 오퍼: 교차 리딩(코어) 59,000원 — 트랙A 한정오퍼

승인 플랜의 매출 엔진은 **카카오 4,000 친구 재활성화**(트랙A). 여기에 코어 59,000을
"가치 선행 + 기간 한정" 오퍼로 투입한다. 입문 19,000~29,000은 콜드 SNS 신규를 데우는
별도 깔때기로 병행(트랙B)하고, 입문 구매자 중 고민이 명확한 사람을 코어로 업셀.

근거(§9): 코어 59,000 기준 일 약 3.4건이면 20만원/일 도달. 재활성 캠페인일엔 단독 달성 가능.
혼합 안전판: 코어 2 + 입문 3 ≈ 18.5~20.5만.

### 배송 플로 (코어)
1. 구글폼으로 생년월일시·성별·고민 1개(연애/직업/이동 중) 수집.
2. sajugen으로 명리+자미두수 교차 PDF 생성(`sajugen/` 파이프라인, CLI/FastAPI 작동 확인됨).
3. 본인이 LLM 보조로 교차 해석·선택 기준 코멘트 작성·**사람 검수**(명식 검수 포함).
4. 입금 확인 → 카톡으로 PDF + 코멘트 전달 → 추가질문 1회 응대.
- 처리시간 목표: 25~40분(템플릿·AI 보조로 단축). 용량은 병목 아님 — 병목은 문의 수.

### 배송 플로 (입문, 콜드 깔때기)
1. 구글폼 수집 → 2. sajugen 미니 PDF → 3. 핵심 1축 코멘트 검수 → 4. 입금 후 전달.
- 처리시간 목표: 15~25분.

## 컴플라이언스 통과 카피 예시 (보장표현 無)

**OK (참고·해석 프레이밍)**
- "당신 명식에서 재물이 어떻게 흐르는지 한 장으로 정리해 드립니다."
- "연애·직업·재물 중 가장 궁금한 1가지를, 명식 근거와 함께 해석해 드려요."
- "막연한 불안 대신, 내 사주를 보는 기준 하나를 가져가세요."

**금지 (보장·단정·자격)**
- ✗ "한 달 안에 정리됩니다 / 반드시 풀립니다 / 100% 적중"
- ✗ "심리상담사 1급이 직접" (자격 명시)
- ✗ "확실히 좋아집니다"

## 가격·마진 메모
- 입문 19,000~29,000: 처리 15~25분, 콜드 리드 확보·업셀 진입용.
- 코어 59,000: 처리 25~40분(AI 보조), 객단가 핵심·트랙A 첫 판매.
- 패키지 149,000+: 7일 내 1건이면 가속. 무리한 푸시 금지.

## 계좌이체 신뢰 카피 (결제 직전 배치 — 보장표현 無)

> 계좌이체는 "입금 후 못 받을까" 불안이 최대 마찰점. 결제 직전에 아래 5요소를
> 한 메시지로 묶어 노출한다(증거→절차→수령물→타임라인→문의처 순).

- **증거 선노출:** "신청 전 [실제 후기]와 [사주풀이 구성]을 먼저 확인하실 수 있어요."
- **입금 절차 명확화:** "입금자명을 신청 시 성함과 동일하게 적어주시면 바로 확인됩니다.
  확인 즉시 '확인되었습니다' 메시지를 보내드려요."
- **수령물 명시:** "받으시는 것: 명리×자미두수 교차 해석 PDF(풀구성) + 추가질문 1회 응대."
- **배송 타임라인:** "입금 확인 후 ○○시간 내 카카오로 PDF를 전달드립니다(영업시간 기준)."
- **문의처·책임:** "전달이 지연되면 이 채팅으로 바로 문의 주세요. 끝까지 책임지고 보내드립니다."

금지: "반드시/100%/한 달 안에/확실히/결과 보장". 허용: 절차·타임라인의 사실적 약속만.
(상호·사업자 정보 노출 범위는 사용자 확정 필요 → 플랜 '승인 시 즉시 필요한 입력' §2.)

## 추가질문 정책 (범위·횟수·응답시간 명시 → 컴플레인 예방)

- **포함 범위:** 전달된 PDF 해석 내용에 대한 이해를 돕는 질문 1회(코어 기준).
  예) "이 부분 타이밍을 제 상황에 맞게 다시 설명해 주세요."
- **제외 범위:** 새 명식 추가 풀이, 무제한 상담, 결과 단정 요구는 별도 오퍼(패키지/딥리딩)로 안내.
- **응답 시간:** 영업시간 기준 ○○시간 내 1회 답변(자동발송 야간차단 시간대는 익일 오전).
- **AI 보조 + 사람 검수:** 추가질문 답변 초안은 AI 보조로 작성하되 **반드시 사람 검수 후 발송**.
- **개인정보:** 생년월일시 등 민감정보는 sajugen 로컬 처리 원칙. 외부 3rd파티 LLM에 불필요 업로드 금지
  (추가질문 보조 워크플로는 privacy-safe 설계 후 도입 → 플랜 §13 Later).
- **표현 가드:** 추가질문 답변에도 결과·시기 보장 금지. "참고용 해석·선택 기준" 프레이밍 유지.
