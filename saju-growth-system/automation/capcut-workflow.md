# CapCut 영상 워크플로 (API 없음 — 템플릿 기반)

> 검증: CapCut은 **공식 공개 렌더/배치/업로드 API가 없다**(2026-06-08).
> 따라서 "프로그램으로 영상 자동 생성·업로드"는 불가. 아래는 현실적 반자동 워크플로.

## 핵심 원칙
- 영상에 큰 시간 쓰지 않는다. **카드뉴스 우선 → 성과 상위만 영상화.**
- "가장 승산 있는 게임" = D3 포맷 A/B로 빠르게 승자 찾고 더블다운.

## 워크플로 A — CapCut 템플릿 (반복 제작)
1. CapCut에서 **사주 릴스 템플릿 1~2종**을 1회 제작(후킹 자막·배경·BGM 슬롯).
2. 매 영상: 자막 텍스트 + 이미지/배경만 교체 → **배치 export**.
3. export한 mp4를 **Metricool**에 업로드 → IG Reels/TikTok/Shorts 예약발행.
4. YouTube 썸네일은 수동 설정(스케줄러 제약).

## 워크플로 B — 카드뉴스 → 릴스 (영상 최소)
1. `content/`에서 카드뉴스(5~7장) 제작(Canva 등).
2. 카드 슬라이드를 **인스타그램 자체 릴스 편집툴** 또는 CapCut에 넣어 자막릴스화.
3. 동일 mp4를 멀티채널 재활용.
