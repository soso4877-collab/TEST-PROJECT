# 리퍼포징 프롬프트 (1 소스 → 11 산출물)

> 소스 후킹 1개를 채널별로 변환. Claude Code 대화 또는 `automation/repurpose.py`에서 사용.
> 공통 제약: 보장·시기 단정 금지, 첫 줄 250자 이하, 원본성 유지(워터마크/재탕 금지).

## 변환 구조
```
소스 아이디어 1개 (반증형 후킹)
 ├─ Threads 3개 (후킹 변형 + 오픈루프 질문)
 ├─ IG 캐러셀 1개 (카드뉴스 5~7장)
 ├─ IG Reel 스크립트 1개 (자막릴스 30초)
 ├─ IG Story 3프레임 (투표/질문 스티커)
 ├─ YouTube Shorts 스크립트 1개
 ├─ TikTok 스크립트 1개
 ├─ 카카오 메시지 1개 ((광고) 표기 + opt-out)
 └─ CTA 3종 (소프트/직접/마감임박)
```

## 마스터 프롬프트 (LLM에 붙여넣기)

```
역할: 사주/명리 콘텐츠 카피라이터. 브랜드 "서담선생".
제약(엄수):
- 결과·시기 보장 표현 금지(표시광고법): "반드시/100% 적중/한 달 안에/확실히" 사용 금지.
- 자격증 명시 금지. 자기자랑·PDF/책자 형식 암시 톤 금지.
- 톤: 2인칭 상담체, 단정 대신 "~일 수 있어요". 첫 줄 250자 이하.
- 원본성: 타 계정/플랫폼 문구 복붙 금지, 고유 관점 추가.

입력 소스 후킹: "{여기에 후킹}"

아래 11개를 생성:
1. Threads 3개 — 각기 다른 후킹 패턴(반증/숫자/자기관련), 끝에 오픈루프 질문 1개.
2. IG 캐러셀 5~7장 — 1장=후킹, 중간=근거/패턴, 마지막=오픈루프+CTA(소프트).
3. IG Reel 30초 스크립트 — [0-3초 후킹][3-20초 본문][20-30초 CTA], 자막 문구 포함.
4. IG Story 3프레임 — 투표/질문 스티커 문구.
5. YouTube Shorts 스크립트 — 키워드 제목(검색용) + 9:16 자막 흐름.
6. TikTok 스크립트 — 트렌드 친화 후킹 변형 + 해시태그 3개.
7. 카카오 (광고) 메시지 — (광고) 표기 자리 + 본문 + 무료 수신거부 안내 자리.
8. CTA 3종 — 소프트/직접/마감임박(시간보장 없는 표현).

출력: 섹션별 마크다운. 각 산출물 끝에 [기둥: 1~4] 태그.
```

## 사용 흐름
1. `content/hooks-library.md`에서 오늘의 후킹 1개 선택.
2. 마스터 프롬프트에 삽입 → LLM 생성 → 검수(컴플라이언스 grep).
3. `content/7-day-content-calendar.csv`에 배치(날짜·채널·상태).
4. `automation/calendar_to_csv.py`로 Publer/Metricool 임포트용 CSV 변환.
